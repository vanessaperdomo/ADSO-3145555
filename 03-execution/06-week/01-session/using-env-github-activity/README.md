# using-env-github-activity

Landing page técnico para explicar la estrategia de ramas por ambientes en el repositorio `using-env-github-activity`.

## Estructura

```text
using-env-github-activity/
├── index.html
├── styles.css
├── script.js
├── README.md
└── assets/
    ├── branch-strategy.svg
    └── hu-workflow.svg
```

## Uso

1. Descomprima el paquete.
2. Abra `index.html` en un navegador.
3. Navegue por las secciones del landing.

## Contenido cubierto

- Contexto del repositorio.
- Política de ambientes `develop`, `qa`, `main`.
- Regla de no tocar ramas padre directamente.
- Flujo de HU por ambiente.
- Comandos ejemplo por rama hija.
- Errores comunes a evitar.
