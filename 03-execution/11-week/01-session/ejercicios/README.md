# Paquete de 5 ejercicios resueltos - Modelo de aerolínea

## Contenido
- `ejercicios_resueltos/`: 5 ejercicios en Markdown, resueltos y explicados paso a paso.
- `scripts_sql/`: scripts SQL de instalación y demostración para cada ejercicio.
- `landing/`: landing page en HTML, CSS y JS con explicación teórica y técnica.

## Orden recomendado de uso
1. Leer el `.md` del ejercicio.
2. Ejecutar el archivo `ejercicio_xx_setup.sql`.
3. Ejecutar el archivo `ejercicio_xx_demo.sql`.
4. Abrir `landing/index.html` para la explicación paso a paso.

## Nota
Los scripts se construyeron usando únicamente tablas y columnas existentes del modelo base.
