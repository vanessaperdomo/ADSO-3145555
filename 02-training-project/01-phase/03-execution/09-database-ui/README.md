# Activity 09: Database & UI / Actividad 09: Base de Datos e IU

## English

### Activity Overview

**Develop data structure and user interface of the software**

This activity implements the database from the data model and creates front-end components while applying environmental protection, occupational safety and health practices.

### Objectives

- Build database from data model
- Create front-end components according to design
- Implement responsive user interfaces
- Apply environmental protection practices
- Ensure workplace safety and health
- Develop technical communication in English
- Integrate SST (Safety and Health at Work) strategies
