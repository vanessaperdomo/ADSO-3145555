# Activity 08: Development / Actividad 08: Desarrollo

## English

### Activity Overview

**Code software modules based on market standards**

This activity focuses on the actual implementation of the software solution through coding, testing, and quality assurance while developing entrepreneurial competencies and exercising fundamental labor rights.

### Objectives

- Plan and execute software construction activities
- Code software according to established design
- Perform comprehensive software testing
- Incorporate quality assurance practices
- Understand fundamental labor rights
- Develop entrepreneurial and business culture
- Apply improvement actions based on results
