# Activity 04: Algorithms / Actividad 04: Algoritmos

## English

### Activity Overview

**Structure logical processes for algorithm construction**

This activity develops computational thinking by designing algorithms and logical processes that solve problems through systematic, mathematical, and scientific approaches.

### Objectives

- Develop logical processes through algorithm implementation
- Apply mathematical and scientific principles to problem-solving
- Create algorithms and pseudocode
- Solve computational problems systematically
- Apply quantitative reasoning to technical contexts
- Verify solutions using technological tools
