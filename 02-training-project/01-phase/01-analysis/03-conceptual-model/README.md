# Activity 03: Conceptual Model / Actividad 03: Modelo Conceptual

## English

### Activity Overview

**Specify the conceptual model of the software**

This activity transforms requirements into visual and conceptual representations through modeling techniques, enabling better understanding and communication of the software solution.

### Objectives

- Model software functions according to requirements
- Create comprehensive UML diagrams
- Verify models against requirements
- Plan analysis activities following methodology
- Develop effective technical communication skills
- Apply analysis best practices

| **240202501** | Interact in English in social and work contexts according to CEFR |
| **240201524** | Develop effective and efficient communication processes in social, personal, and productive contexts |
