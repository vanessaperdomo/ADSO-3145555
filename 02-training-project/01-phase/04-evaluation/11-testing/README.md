# Activity 11: Testing / Actividad 11: Pruebas

## English

### Activity Overview

**Perform quality verification activities for the software**

This evaluation activity focuses on comprehensive quality assurance through systematic testing methodologies, verification processes, and continuous improvement activities to ensure the software meets all requirements and quality standards.

### Objectives

- Execute systematic testing procedures
- Verify software quality against requirements
- Identify and document defects
- Implement quality improvement actions
- Validate functionality and performance
- Ensure user satisfaction
- Apply quality assurance methodologies

3. **System Testing**

4. **Acceptance Testing**

### Objetivos

- Ejecutar procedimientos sistemáticos de pruebas
- Verificar calidad de software contra requisitos
- Identificar y documentar defectos
- Implementar acciones de mejora de calidad
- Validar funcionalidad y rendimiento
- Asegurar satisfacción del usuario
- Aplicar metodologías de aseguramiento de calidad
