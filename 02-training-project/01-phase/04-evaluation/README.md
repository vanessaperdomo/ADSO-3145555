# Phase 04: Evaluation / Fase 04: Evaluación

## English

### Overview

The **Evaluation Phase** is the final stage where acquired competencies are demonstrated and software quality is rigorously verified. This phase ensures the solution meets all requirements and quality standards.

### Objectives

- Demonstrate acquired competencies through practical application
- Verify software quality against industry standards
- Conduct comprehensive testing and quality assurance
- Validate learning outcomes achievement
- Prepare for professional practice stage

This phase contains **2 project activities**:

| # | Activity | Focus Area |

### Deliverables

## Español

### Descripción General

La **Fase de Evaluación** es la etapa final donde se demuestran las competencias adquiridas y se verifica rigurosamente la calidad del software. Esta fase asegura que la solución cumple con todos los requisitos y estándares de calidad.

### Objetivos

- Demostrar competencias adquiridas mediante aplicación práctica
- Verificar calidad del software según estándares de la industria
- Conducir pruebas comprensivas y aseguramiento de calidad
- Validar el logro de resultados de aprendizaje
- Preparar para la etapa de práctica profesional

### Actividades

Esta fase contiene **2 actividades de proyecto**:

| # | Actividad | Área de Enfoque |
|---|-----------|-----------------|
| 11 | [Pruebas](11-testing) | Verificación de calidad y testing |
| 12 | [Evaluación](12-evaluation) | Demostración de competencias |

### Competencias Clave

- **999999999** - Resultados de aprendizaje etapa práctica
- **220501098** - Controlar calidad del servicio de software según estándares técnicos

### Resultados de Aprendizaje (RAPs)

Los resultados de aprendizaje clave incluyen:
- **590803** - Aplicar en la resolución de problemas reales del sector productivo, los conocimientos, habilidades y destrezas pertinentes a las competencias del programa asumiendo estrategias y metodologías de autogestión
- **593144-02** - Verificar la calidad del software de acuerdo con las prácticas asociadas en los procesos de desarrollo
- **593145-03** - Realizar actividades de mejora de la calidad del software a partir de los resultados de la verificación

### Entregables

- Plan de pruebas comprensivo
- Casos y escenarios de prueba
- Informes de aseguramiento de calidad
- Registros de seguimiento y resolución de bugs
- Resultados de pruebas de rendimiento

- Informe de evaluación final
- Portafolio de demostración de competencias
- Documentos de preparación para etapa práctica

### Métricas de Calidad

- Porcentaje de cobertura de código
- Distribución de severidad de bugs
- Benchmarks de rendimiento
- Puntuaciones de aceptación de usuario
- Cumplimiento con estándares de calidad
- Resultados de evaluación de seguridad
