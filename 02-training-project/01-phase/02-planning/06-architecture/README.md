# Activity 06: Architecture / Actividad 06: Arquitectura

## English

### Activity Overview

**Determine the logical and technological structure of the software**

This activity establishes the software's architecture by defining the data model, system design artifacts, and conducting formative research to support technical decisions.

### Objectives

- Structure software data model according to analysis specifications
- Elaborate design artifacts following methodology practices
- Verify design phase deliverables
- Conduct formative research for informed decisions
- Analyze productive context characteristics
- Discuss solutions in social and work contexts

#### 02-learning-activity
