# Activity 07: Technical Proposal / Actividad 07: Propuesta Técnica

## English

### Activity Overview

**Structure the technical proposal for the software project**

This activity consolidates all planning work into a comprehensive technical proposal that defines specifications, validates conditions with stakeholders, and establishes ethical frameworks for project execution.

### Objectives

- Define complete technical specifications
- Elaborate comprehensive technical proposal
- Validate proposal with all stakeholders
- Establish ethical principles for project execution
- Promote sustainable development practices
- Build consensus for peace culture

### Associated Competencies

| Code | Competency |
|------|------------|
| **220501094** | Structure technical proposal for IT services according to technical requirements and regulations |
| **240201526** | Interact in productive and social contexts according to ethical principles for building a culture of peace |

### Learning Outcomes (RAPs)

| Code | Learning Outcome |
|------|------------------|
| **593060-01** | Define software technical specifications according to software characteristics to be built |
| **593061-03** | Validate technical proposal conditions according to stakeholder interests |
| **593062-02** | Elaborate software technical proposal according to defined technical specifications |
| **593147-02** | Establish personal and community growth relationships from common good as contribution to social development |
| **593148-03** | Promote rational use of natural resources from sustainability and ethical and regulatory sustainability criteria |
| **593149-01** | Promote my dignity and others' from ethical principles and values as contribution to establishing a culture of peace |
| **593150-04** | Contribute to strengthening peace culture from human dignity and strategies for conflict transformation |
