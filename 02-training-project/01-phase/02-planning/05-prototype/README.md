# Activity 05: Prototype / Actividad 05: Prototipo

## English

### Activity Overview

**Build software prototype according to functional and quality characteristics analysis**

This activity focuses on creating interactive prototypes that demonstrate the user interface, user experience, and key functionalities of the software solution while promoting healthy work habits.

### Objectives

- Design and build functional prototypes
- Determine technical characteristics of GUI
- Validate design with stakeholders
- Apply UI/UX best practices
- Present work processes effectively
- Promote healthy lifestyle habits in productive contexts

|------|------------|
| **220501095** | Design software solution according to procedures and technical requirements |
| **240202501** | Interact in English in social and work contexts according to CEFR |
| **230101507** | Generate healthy lifestyle habits through physical activity programs in productive and social contexts |

### Learning Outcomes (RAPs)

| Code | Learning Outcome |
|------|------------------|
| **593100-03** | Determine technical characteristics of software GUI adopting standards |
| **593116-05** | Present a process for performing an activity in their work according to procedures established in their training program |

| **593120-01** | Develop psychomotor skills in productive and social contexts |
| **593121-03** | Execute physical conditioning activities oriented toward improving physical condition in productive and social contexts |
| **593122-04** | Implement ergonomics and active breaks plan according to function characteristics |

## Español

### Descripción de la Actividad

**Construir el prototipo del software de acuerdo con el análisis de las características funcionales y de calidad**

Esta actividad se enfoca en crear prototipos interactivos que demuestran la interfaz de usuario, experiencia de usuario y funcionalidades clave de la solución de software mientras se promueven hábitos de trabajo saludables.

### Objetivos

- Diseñar y construir prototipos funcionales
- Determinar características técnicas de la GUI
- Validar diseño con interesados
- Aplicar mejores prácticas de UI/UX
- Presentar procesos de trabajo efectivamente
- Promover hábitos de vida saludable en contextos productivos
