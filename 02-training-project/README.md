## 📋 Estructura del Proyecto Formativo

El proyecto formativo está organizado jerárquicamente de la siguiente manera:

```
FASES (4)
  └── ACTIVIDADES DE PROYECTO (12)
       └── COMPETENCIAS (múltiples por actividad)
            └── RESULTADOS DE APRENDIZAJE - RAPs (múltiples por competencia)
```

### Jerarquía del Proceso Formativo

1. **FASES**: Grandes etapas del ciclo de desarrollo de software
2. **ACTIVIDADES DE PROYECTO**: Tareas específicas que se ejecutan en cada fase
3. **COMPETENCIAS**: Capacidades que se desarrollan a través de las actividades
4. **RAPs (Resultados de Aprendizaje)**: Conocimientos, habilidades y destrezas específicas a lograr
